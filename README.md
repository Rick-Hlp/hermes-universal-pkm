# Hermes Universal PKM

**Universal Personal Knowledge Management** + **Hermes/Obsidian Bidirectional Sync**

整合版知识库框架：统一目录结构 + 双向同步 + 智能分类

---

## 🎯 核心理念

```
任何输入 → 智能分类 → 统一归档 → 双向同步
```

- **Universal PKM**：一套目录结构，任何内容都能归档
- **Hermes Sync**：Hermes/OpenClaw 多实例自动同步
- **统一整合**：技能文档、问题解决、资源收集都在同一框架

---

## 📦 目录结构

```
笔记文档/
├── 00-收件箱/                    # 快速捕获
├── 10-资产/                      # 数字资产
│   ├── 账号/
│   ├── 订阅/
│   └── 设备/
├── 20-资源/                      # 外部资源 ⭐
│   ├── 技能/                     #   技能库
│   │   └── {技能名}/             #     每个技能独立目录
│   │       ├── 指南/             #       使用指南
│   │       ├── 问题/             #       问题与解决
│   │       ├── 优化/             #       优化记录
│   │       └── 备份/             #       备份策略
│   ├── 项目/
│   ├── 工具/
│   └── 文章/
├── 30-想法/                      # 内部想法
│   ├── 闪念/
│   └── 洞察/
├── 40-项目/                      # 当前项目
├── 50-领域/                      # 责任领域
├── 60-归档/                      # 归档
├── 90-地图/                      # 地图导航
└── 99-系统/                      # 模板和配置
```

## 🚀 核心特性

| 特性 | 说明 |
|-----|------|
| **四大核心库** | 资产 + 资源 + 想法 + 项目 |
| **技能子目录** | 每个技能有自己的 指南/问题/优化 |
| **双向同步** | Hermes/OpenClaw ↔ Obsidian 自动同步 |
| **变更记录** | 每次修改强制记录来源和类型 |
| **智能分类** | AI 自动识别内容类型并归档 |
| **快速命令** | `/技能` `/问题` `/想法` `/账号` 等快捷指令 |

## 🚀 快速开始

### 方式 1：直接发给我（推荐）

微信/飞书发任何内容，我自动识别、自动归档。

```
你：发现个好项目 https://github.com/xxx 能自动同步 Obsidian
我：→ 自动创建 20-资源/项目/xxx.md

你：GitHub Pro 订阅到期 2026-12，$4/月
我：→ 自动创建 10-资产/订阅/github-pro.md

你：突然想到应该监控 Kickstarter
我：→ 自动创建 30-想法/闪念/kickstarter-monitor.md
```

### 方式 2：快捷命令

| 命令 | 动作 | 目标位置 |
|-----|------|---------|
| `/账号` | 添加账号 | 10-资产/账号/ |
| `/订阅` | 添加订阅 | 10-资产/订阅/ |
| `/资源` | 添加资源 | 20-资源/{类别}/ |
| `/技能` | 添加技能 | 20-资源/技能/{名称}/ |
| `/问题` | 记录问题 | 20-资源/技能/{名称}/问题/ |
| `/想法` | 记录想法 | 30-想法/闪念/ |
| `/项目` | 创建项目 | 40-项目/{名称}/ |
